在纠结了很久的发行版选择后，最终还是选择了ubuntu16.04LTS。可能人老了就是这样吧。但再怎么说ubuntu也是linux，还是不得不折腾的。。。

折腾的东西很多很杂，以后可能忘了，所以记录一下。

第一步：安装ubuntu

  1. 下载ubuntu镜像，我这里下载不知为何10+mb/s，而且是官网
  2. 准备一个2g以上的u盘，安装好ultraiso
  3. 写入磁盘映像
  4. 在win下预留空间。打开磁盘管理，压缩卷。这里我压出了100G。
  5. 重启，进bios改引导。这里要看一看bios，把安全启动（secure boot）和快速启动都关掉，csm模块全部改成uefi优先。
  6. 开始安装，安装程序检测到了linux，但是这里不要选共存，选“其他选项”，手动分区
  7. 我的打算是70个g给linux，剩下30g使用exfat，这样两个系统都能操作。于是，直接把70个g挂载到根分区上（简单粗暴）。这里注意启动项要选在win10的efi分区上。
  8. 密码、时区之类的设置，然后就是等待了

第二步：更新系统（可以先修改软件源）

sudo apt-get update
sudo apt-get upgrade
sudo apt-get dist-upgrade

第三步：检测并安装驱动

打开软件与更新–>附加驱动

第四步：基本的美化

安装tweak-tool

sudo apt-get install unity-tweak-tool

做自己喜欢的调整，主题我用了arc，图标是numix-circle

第五步：安装显卡驱动

我这张卡是amd rx480,在官网找到驱动下载即可，按提示安装即可。

如果要卸载：

sudo amdgpu-pro-uninstall
